## Installation -
 - Clone the project
 - cd react-login
 - npm install
 - cd client
 - npm install
 - cd ..
 - npm start




 


## Run Application -
- Run - 
  > npm start
- Duplicate current tab to add as many participants you want

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

Socket server will be running on http://localhost:8000

## Used -
- Node.js
- Socket.IO
- React.js (Hooks)
